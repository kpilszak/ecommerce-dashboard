## General info
This project is a Spring application.  
The main purpose is to develop admin dashboard for e-commerce.  

## Technologies
Project is created with:
* Java  
* Spring Framework 5
* Spring Boot 2
* Spring MVC
* Spring Data
* Hibernate
* Thymeleaf  
  
IDE: Intellij IDEA 2020
